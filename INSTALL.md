# GNU MAKE

### Installation Instructions:

- **Arch Linux / Manjaro**:  
  `sudo pacman -S make`

- **Fedora**:  
  `sudo dnf install make`

- **CentOS / RHEL**:  
  `sudo yum install make`

- **Ubuntu**:  
  `sudo apt install make`
